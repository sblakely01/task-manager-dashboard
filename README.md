### Features

#### Field Validation:

- Utilizes input validity object to identify when required feilds are not filled out

#### Tasks Saving:
- Tasks are saved in localStorage and any changes to existing tasks are immediately updated via the useEffect hook

#### Filtering and Searching:
- Filtering and searching functions are included to sort through all tasks
